## üéØ EXECUTIVE SCOPE (10-15 min)

### Why Social Media Metrics Matter

**Key Insight:** Social media products are about **network effects** and **engagement loops**. Metrics must capture both user value (connection, content) and business value (ad revenue, growth).

**Business Model Characteristics:**
- Two-sided: creators + consumers
- Network effects: value increases with users
- Engagement loops: content ‚Üí engagement ‚Üí more content
- Monetization: ads, premium features, creator tools

### BUCKET 1: Define NSM for Social Media Products
**Questions:** ~15 | **Priority:** üü¢ GREEN (Master this)

**Board Slide Bullets:**
- **NSM Options:** DAU/MAU (growth), Time Spent (engagement), Weekly Engaged Users (balanced)
- **Key Insight:** Must balance creator value (content creation) + consumer value (content consumption)
- **Input KPIs:** Engagement (likes, comments, shares), Content Creation (posts, stories), Network Effects (connections, messages)
- **Leading Indicators:** First-week retention, Time to first engagement, Content upload rate
- **Guardrails:** User safety (reports, blocks), Content moderation, Creator burnout

**Concrete Examples:**
- "Instagram NSM: Weekly Engaged Users (users who post or engage weekly) - balances creation + consumption"
- "TikTok NSM: Daily Video Views per Active User - content consumption + engagement"
- "Facebook NSM: Daily Active Users - network effects + engagement"

**Representative Question Types (Do 5 only):**
- Define NSM for Instagram/TikTok/Facebook
- What metrics would you track for a new social media feature?
- How would you measure success of Stories/Reels/Posts?
- What are the key metrics for a social platform?
- How would you define success for a creator platform?

**‚ù§Ô∏è Reusable Narrative (Base Story - Adapt for Each Question):**

**Framework:** `Balance Creator + Consumer Value ‚Üí NSM ‚Üí Input KPIs ‚Üí Leading Indicators ‚Üí Guardrails`

**Memorizable Answer:**

For social media products, the NSM must balance creator value and consumer value.

**1Ô∏è‚É£ Define NSM** ‚Üí Typically Weekly Engaged Users - users who either create content or engage with content weekly. Captures network effects and engagement loops.

**2Ô∏è‚É£ Input KPIs** ‚Üí Engagement rate (likes, comments, shares per user), content creation rate (posts/stories per user), network effects (connections made, messages sent).

**3Ô∏è‚É£ Leading Indicators** ‚Üí First-week retention, time to first engagement, content upload rate.

**4Ô∏è‚É£ Guardrails** ‚Üí User safety (reports, blocks), content moderation effectiveness, creator satisfaction (prevent burnout).

**Key Principle:** Balance creator value (content creation) and consumer value (content consumption) - both sides must win.

### BUCKET 2: Engagement & Content Metrics
**Questions:** ~12 | **Priority:** üü¢ GREEN (High-yield)

**Board Slide Bullets:**
- **Engagement Metrics:** Likes, comments, shares, time spent, session frequency
- **Content Metrics:** Posts created, content quality, viral coefficient
- **Key Insight:** Engagement drives retention, content drives engagement
- **Measurement:** Engagement rate = (likes + comments + shares) / impressions
- **Quality Signals:** Completion rate, save rate, share rate

**Concrete Examples:**
- "Engagement rate: (likes + comments + shares) / impressions - measures content resonance"
- "Content creation rate: posts per active user - measures creator health"
- "Viral coefficient: shares per post - measures network effects strength"

**Representative Question Types (Do 5 only):**
- How would you measure engagement on a social platform?
- What metrics indicate content quality?
- How do you measure network effects?
- What's the difference between reach and engagement?
- How would you improve content creation rates?

**‚ù§Ô∏è Reusable Narrative (Base Story - Adapt for Each Question):**

**Framework:** `Measure Breadth + Depth ‚Üí Engagement Rate ‚Üí Content Quality ‚Üí Network Effects`

**Memorizable Answer:**

For engagement metrics, I measure both breadth (how many users engage) and depth (how much they engage).

**1Ô∏è‚É£ Engagement Rate** ‚Üí (likes + comments + shares) / impressions.

**2Ô∏è‚É£ Engagement Depth** ‚Üí Time spent, session frequency, content creation rate.

**3Ô∏è‚É£ Content Quality** ‚Üí Completion rate, save rate, share rate - these indicate content value.

**4Ô∏è‚É£ Network Effects** ‚Üí Connections made, messages sent, group participation.

**Key Principle:** Balance creator metrics (content creation) with consumer metrics (engagement) to maintain a healthy ecosystem.

### BUCKET 3: Creator Platform Metrics
**Questions:** ~10 | **Priority:** üü° YELLOW

**Board Slide Bullets:**
- **Creator Metrics:** Creator retention, earnings, satisfaction, content quality

- What metrics matter for a creator platform?
- How do you prevent creator burnout?
- What's the relationship between creator metrics and platform metrics?

**‚ù§Ô∏è Reusable Narrative (Base Story - Adapt for Each Question):**

**Framework:** `Creator Health ‚Üí Retention + Earnings + Satisfaction ‚Üí Input KPIs ‚Üí Guardrails`

**Memorizable Answer:**

For creator platforms, I measure creator health.

**1Ô∏è‚É£ Creator Health** ‚Üí Retention (creators who post monthly), earnings (revenue per creator), satisfaction (NPS or satisfaction score).

**2Ô∏è‚É£ Input KPIs** ‚Üí Content creation rate, content quality score, creator growth rate.
