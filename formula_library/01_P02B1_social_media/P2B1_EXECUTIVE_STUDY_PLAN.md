### Why Social Media Metrics Matter

**Key Insight:** Social media products are about **network effects** and **engagement loops**. Metrics must capture both user value (connection, content) and business value (ad revenue, growth).

**Business Model Characteristics:**
- Two-sided: creators + consumers
- Network effects: value increases with users
- Engagement loops: content â†’ engagement â†’ more content
- Monetization: ads, premium features, creator tools

### BUCKET 2: Engagement & Content Metrics
**Questions:** ~12 | **Priority:** ğŸŸ¢ GREEN (High-yield)

**Board Slide Bullets:**
- **Engagement Metrics:** Likes, comments, shares, time spent, session frequency
- **Content Metrics:** Posts created, content quality, viral coefficient
- **Key Insight:** Engagement drives retention, content drives engagement
- **Measurement:** Engagement rate = (likes + comments + shares) / impressions
- **Quality Signals:** Completion rate, save rate, share rate

**Concrete Examples:**
- "Engagement rate: (likes + comments + shares) / impressions - measures content resonance"
- "Content creation rate: posts per active user - measures creator health"
- "Viral coefficient: shares per post - measures network effects strength"

**Representative Question Types (Do 5 only):**
- How would you measure engagement on a social platform?
- What metrics indicate content quality?
- How do you measure network effects?
- What's the difference between reach and engagement?
- How would you improve content creation rates?

**â¤ï¸ Reusable Narrative (Base Story - Adapt for Each Question):**
> "For engagement metrics, I measure both breadth (how many users engage) and depth (how much they engage). Engagement rate = (likes + comments + shares) / impressions. I also track time spent, session frequency, and content creation rate. For content quality, I use completion rate, save rate, and share rate - these indicate content value. Network effects are measured by connections made, messages sent, and group participation. The key is balancing creator metrics (content creation) with consumer metrics (engagement) to maintain a healthy ecosystem."
