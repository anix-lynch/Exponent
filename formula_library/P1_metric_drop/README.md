## Overview

P1 is for diagnosing metric drops - when "X is down Y%" questions come up.

**Core Flow:**
1. Clarify what metric dropped
2. Check if it's a data issue
3. Segment to find the hot spot
4. Hypothesize why
5. Validate hypotheses
6. Take action
