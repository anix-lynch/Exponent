## üìä CONCEPT BUCKET BREAKDOWN

### BUCKET 1: Content Platform Design
**Questions:** ~15 | **Priority:** üü¢ GREEN (Master this)

**Board Slide Bullets:**
- **What:** "Design a video streaming app" or "Design a content platform" - core content platform framework
- **Framework:** Users ‚Üí Jobs ‚Üí Core Loop ‚Üí Key Features ‚Üí Tradeoffs ‚Üí Success Metrics
- **Users:** Who are the users? (Content creators, Viewers/consumers, Advertisers if applicable), What are their needs? (Content discovery, Quality content, Monetization for creators)
- **Jobs:** What jobs do users need done? (Discover content, Consume content, Create content, Monetize content), Job priority (Frequency, Importance)
- **Core Loop:** What is the engagement loop? (Discover content ‚Üí Watch/consume ‚Üí Engage (like, comment) ‚Üí Return for more), How does it drive retention? (Content library, Personalized recommendations)
- **Key Features:** Core features (Content library, Recommendation engine, Player/viewer, Creator tools), Supporting features (Search, Playlists, Social features)
- **Tradeoffs:** Content quality vs quantity, Creator vs viewer experience, Free vs paid content
- **Success Metrics:** Consumption metrics (Watch time, Completion rate, Sessions per user), Creator metrics (Creator satisfaction, Content upload rate)

**Concrete Examples:**
- "Design video streaming: Users (Content creators, Viewers, Advertisers), Jobs (Discover, watch, create, monetize), Core Loop (Discover ‚Üí Watch ‚Üí Engage ‚Üí Return), Key Features (Library, recommendations, player, creator tools), Tradeoffs (Quality vs quantity, Creator vs viewer), Success Metrics (Watch time, completion rate, creator satisfaction)"
- "Design podcast platform: Users (Podcast creators, Listeners), Jobs (Discover podcasts, Listen, Create, Monetize), Core Loop (Discover ‚Üí Listen ‚Üí Subscribe ‚Üí Return), Key Features (Library, recommendations, player, creator tools), Tradeoffs (Quality vs quantity, Creator vs listener), Success Metrics (Listen time, subscriptions, creator satisfaction)"
