## üìä CONCEPT BUCKET BREAKDOWN

### BUCKET 1: Revenue Levers (Price, Volume, Mix)
**Questions:** ~40 | **Priority:** üü¢ GREEN (Master this)

**Board Slide Bullets:**
- **What:** "How would you increase revenue?" or "How would you optimize revenue?" - core revenue optimization framework
- **Framework:** Levers (Price, Volume, Mix) ‚Üí Test ‚Üí Measure ‚Üí Iterate
- **Price Lever:** Pricing strategy (Increase price, Optimize pricing tiers, Dynamic pricing), Price elasticity (Test price sensitivity, Find optimal price point)
- **Volume Lever:** User acquisition (Increase new users, Improve conversion funnel), Engagement (Increase usage frequency, Session depth, Feature adoption), Retention (Reduce churn, Improve lifetime value)
- **Mix Lever:** Product mix (Optimize product portfolio, Cross-sell, Upsell), Customer mix (Focus on high-value segments, Improve ARPU), Channel mix (Optimize acquisition channels, Improve CAC efficiency)
- **Test:** Design experiments (A/B tests, Phased rollouts, Market tests), Test one lever at a time (Isolate impact), Control for external factors (Seasonality, Market conditions, Statistical significance)
- **Measure:** Primary metric (Revenue impact - Total revenue, ARPU, LTV), Secondary metrics (Conversion rate, Retention, Engagement), Guardrails (Monitor churn, User satisfaction, Brand impact), Time horizon (Short-term vs long-term revenue impact)
- **Iterate:** If successful (Scale the change, Expand to more segments), If mixed (Refine the lever, Test variations), If unsuccessful (Try different lever, Pivot strategy)

**Concrete Examples:**
- "Increase revenue: Levers (Price: optimize pricing tiers, Volume: improve retention, Mix: focus on high-value customers), Test (A/B test pricing, phased rollout), Measure (Revenue impact, ARPU, churn), Iterate (Scale if successful, refine if mixed)"
- "10x revenue: Levers (Price: dynamic pricing, Volume: 10x user base, Mix: optimize product mix), Test (Market tests, phased approach), Measure (Revenue, retention, satisfaction), Iterate (Scale winners, pivot losers)"
