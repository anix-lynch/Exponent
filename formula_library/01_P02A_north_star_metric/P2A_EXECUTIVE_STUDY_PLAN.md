## üîç HOW TO IDENTIFY P2A (NORTH STAR METRIC) QUESTIONS

**Even when "north star" isn't mentioned, look for these keywords/phrases:**

### Explicit NSM Keywords:
- "north star metric", "NSM", "single metric", "primary metric", "most important metric"
- "define success", "what metric matters most", "key metric"

### Implicit P2A Indicators:
- **NSM questions:** "What is the north star metric for X?", "Define the primary metric", "What's the most important metric?"
- **Success definition:** "How do you define success?", "What metric represents success?"
- **Single metric focus:** "If you could only track one metric, what would it be?"

### P2A vs P2 Distinction:
- **P2A (North Star Metric):** "What is the north star metric?" ‚Üí Focus: Single metric definition
- **P2 (NSM + KPI Ladder):** "What metrics for this product?" ‚Üí Focus: Full KPI ladder (NSM + Input KPIs + Guardrails)

### Red Flags (NOT P2A):
- "What metrics for this product?" ‚Üí P2 (NSM + KPI Ladder)
- "Orders down 25%" ‚Üí P1 (Metric Drop)

## üìä CONCEPT BUCKET BREAKDOWN

### BUCKET 1: North Star Metric Definition
**Questions:** ~50 | **Priority:** üü¢ GREEN (Master this)

**Board Slide Bullets:**
- **What:** "What is the north star metric for [Product]?" - core NSM definition framework
- **Framework:** Define NSM ‚Üí Input KPIs ‚Üí Leading Indicators ‚Üí Guardrails
- **Define NSM:** What is THE single metric? (User value: What do users care about?, Business value: What drives business?, Alignment: How do they connect?), Why this metric? (Actionable: Can we influence it?, Measurable: Can we track it?, Predictive: Does it predict long-term success?)
- **Input KPIs:** What drives the NSM? (User behavior drivers, Supply/quality drivers, Monetization drivers), Controllable levers (What can we change?, What moves the NSM?)
- **Leading Indicators:** Early signals (Short-term behaviors, Funnel entry/activation signals, Quality proxies), Why leading? (Faster feedback, Predict NSM movement)
- **Guardrails:** What must NOT break? (User trust, Long-term health, Cost/abuse/compliance), Safety metrics (Prevent gaming, Protect against unintended consequences)

**Concrete Examples:**
- "Airbnb NSM: Nights booked (user value = travel experience, business value = revenue, alignment = both benefit from bookings)"
- "Instagram NSM: Daily active users who share (user value = expression, business value = engagement, alignment = sharing drives growth)"
- "Uber NSM: Completed rides per week (user value = mobility, business value = marketplace health, alignment = rides = value)"

When defining a north star metric, I use Define NSM ‚Üí Input KPIs ‚Üí Leading Indicators ‚Üí Guardrails.

**1Ô∏è‚É£ Define NSM** ‚Üí What is THE single metric?
  - **User value:** What do users care about?
  - **Business value:** What drives business?
  - **Alignment:** How do they connect?

**Why this metric?**
  - **Actionable:** Can we influence it?
  - **Measurable:** Can we track it?
  - **Predictive:** Does it predict long-term success?

**2Ô∏è‚É£ Input KPIs** ‚Üí What drives the NSM?
  - **User behavior drivers:** What user actions move the NSM?
  - **Supply/quality drivers:** What supply/quality factors move it?
  - **Monetization drivers:** What revenue factors move it?

**Controllable levers:** What can we change? What moves the NSM?

**3Ô∏è‚É£ Leading Indicators** ‚Üí Early signals:
  - **Short-term behaviors:** Actions that predict NSM
  - **Funnel entry/activation signals:** Early engagement indicators
  - **Quality proxies:** Quality metrics that predict NSM

**Why leading?** Faster feedback, predict NSM movement.

**4Ô∏è‚É£ Guardrails** ‚Üí What must NOT break?
  - **User trust:** Metrics that protect user trust
  - **Long-term health:** Metrics that ensure sustainability
  - **Cost/abuse/compliance:** Metrics that prevent abuse

**Safety metrics:** Prevent gaming, protect against unintended consequences.

**Key Principle:** One NSM that represents both user value and business value, actionable and predictive.

**How to Adapt This Narrative for Each Question:**

- **Q188 (Airbnb NSM):** Focus on marketplace NSM
  - "Define NSM: User value (guests want great stays, hosts want bookings), Business value (revenue from bookings, marketplace health), Alignment (bookings = both benefit), Why this metric (Actionable: we can influence bookings, Measurable: track bookings, Predictive: predicts long-term success)"
  - "NSM: Nights booked (captures both guest value and host value, drives revenue, actionable)"
  - "Input KPIs: User behavior (search ‚Üí book ‚Üí stay), Supply (host listings, quality), Monetization (booking fees, host fees)"
  - "Leading Indicators: Searches, booking intent, host response rate"
  - "Guardrails: User trust (safety, reviews), Long-term health (host satisfaction, guest retention), Cost (fraud, abuse)"

- **Q193 (TikTok NSM):** Focus on content platform NSM
  - "Define NSM: User value (entertainment, discovery, creation), Business value (engagement, ad revenue, growth), Alignment (time spent = engagement = revenue), Why this metric (Actionable: we can influence engagement, Measurable: track time spent, Predictive: predicts retention and revenue)"
  - "NSM: Daily active users who watch 30+ minutes (engagement + retention)"
  - "Input KPIs: User behavior (watch time, shares, creates), Content (creator uploads, quality), Monetization (ad views, engagement)"
  - "Leading Indicators: Video starts, completion rate, shares"
  - "Guardrails: User trust (content safety, privacy), Long-term health (creator satisfaction, user retention), Cost (content moderation, abuse)"
