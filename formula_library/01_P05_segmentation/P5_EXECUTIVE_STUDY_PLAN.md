## üîç HOW TO IDENTIFY P5 (SEGMENTATION) QUESTIONS

**Even when "segmentation" isn't mentioned, look for these keywords/phrases:**

### Explicit Segmentation Keywords:
- "segment", "segmentation", "persona", "user groups", "customer segments"
- "which users", "who should we focus on", "target audience"

### Implicit P5 Indicators:
- **User focus questions:** "which users should we focus on?", "who is most valuable?", "different users behave differently"
- **Persona questions:** "create buyer personas", "understand customers", "core audiences"
- **Market questions:** "which market segment?", "which geography?", "target segment"
- **Behavior questions:** "segment by behavior", "engagement by segment", "churn by segment"
- **Design for segments:** "design for X segment", "tailor for Y users", "customize for Z"

### P5 vs P1 Distinction:
- **P5 (Segmentation):** "Which users should we focus on?" ‚Üí Focus: DEFINE segments (Persona √ó Behavior √ó Value)
- **P1 (Metric Drop):** "Orders down 25%" ‚Üí Focus: DIAGNOSE why metric dropped (segment to find hot spot)

### P5 vs P2 Distinction:
- **P5 (Segmentation):** "Which users matter most?" ‚Üí Focus: DEFINE and RANK segments
- **P2 (NSM + KPI):** "What metrics for this product?" ‚Üí Focus: DEFINE what to measure

### P5 vs P4 Distinction:
- **P5 (Segmentation):** "Segment users for churn analysis" ‚Üí Focus: DEFINE segments to analyze
- **P4 (Cohort):** "Retention dropped 35%" ‚Üí Focus: COMPARE cohorts over time

### Red Flags (NOT P5):
- "Orders down 25%" ‚Üí P1 (Metric Drop)
- "Users drop at checkout" ‚Üí P3 (Funnel)
- "Define success metrics" ‚Üí P2 (NSM + KPI)

## üìä CONCEPT BUCKET BREAKDOWN

### BUCKET 1: Classic User/Customer Segmentation
**Questions:** ~5 | **Priority:** üü¢ GREEN (Master this)

**Board Slide Bullets:**
- **What:** "Which users should we focus on?" or "How to segment customers?" - core segmentation framework
- **Framework:** (Persona √ó Behavior √ó Value) ‚Üí Rank ‚Üí Focus Top Segments
- **Persona (WHO):** Demographics (age, role, company size), Context (job-to-be-done, environment), Needs/constraints
- **Behavior (HOW):** Frequency (daily/weekly/occasional), Depth (light vs power usage), Lifecycle stage (new/active/mature)
- **Value (WHY):** Revenue (ARPU, LTV), Strategic value (growth, network effects), Cost/risk (support, churn risk)
- **Rank & Focus:** Impact on core metric, Size/growth potential, Effort to serve ‚Üí Pick primary segment

**Concrete Examples:**
- "Segment e-commerce users: Persona (students, professionals, families) √ó Behavior (browsers, buyers, power shoppers) √ó Value (low/medium/high LTV) ‚Üí Rank ‚Üí Focus on power shoppers"
- "Create buyer personas: Define demographics, context, needs ‚Üí Map behaviors ‚Üí Assess value ‚Üí Rank segments"
- "Segment for growth: Identify high-value segments (high LTV, low churn) ‚Üí Focus product/features on them"

**Representative Questions (Do 5 only):**
- Q153: Can you provide details on your strategies or methods for understanding customers or core audiences?
- Q1069: How do you create buyer personas?
- Q1559: How would you segment users for a new e-commerce platform?
- Q2147: Tell me how you'd segment customers to support a growth initiative.
- Q690: Design Whatsapp for either one of 3 segments: senior citizens, families, or businesses.

**‚ù§Ô∏è Reusable Narrative (Base Story - Adapt for Each Question):**

**Framework:** `Persona (WHO) √ó Behavior (HOW) √ó Value (WHY) ‚Üí Rank ‚Üí Focus`

**Memorizable Answer:**

When I need to segment users, I use the Persona √ó Behavior √ó Value framework.

**1Ô∏è‚É£ Define Personas (WHO)** ‚Üí Demographics (age, role, company size), context (job-to-be-done, environment).

**2Ô∏è‚É£ Define Behaviors (HOW)** ‚Üí Frequency of use (daily/weekly/occasional), depth of usage (light vs power users), lifecycle stage (new/active/mature).

**3Ô∏è‚É£ Assess Value (WHY)** ‚Üí Revenue metrics (ARPU, LTV), strategic value (growth potential, network effects), cost/risk (support burden, churn risk).

**4Ô∏è‚É£ Combine into Segments** ‚Üí Persona √ó Behavior √ó Value.

**5Ô∏è‚É£ Rank Segments** ‚Üí By impact on core metric, size/growth potential, effort to serve.

**6Ô∏è‚É£ Focus** ‚Üí Top segment as primary design target, secondary segment as nice-to-have, deprioritize the rest.

**Key Principle:** Segmentation = WHO they are √ó HOW they behave √ó WHY they matter.

---

**How to Adapt This Narrative for Each Question:**

- **Q153 (Understanding customers/core audiences):** Focus on research methods
  - "Define Personas through user research: interviews, surveys, data analysis to identify demographics, context, needs"
  - "Analyze Behaviors: usage frequency, depth, lifecycle stage from product analytics"
  - "Assess Value: revenue metrics (ARPU, LTV), strategic value (growth, network effects)"
  - "Combine into segments and rank to identify core audience that matters most"

- **Q1069 (Create buyer personas):** Emphasize persona creation
  - "Start with Personas: demographics (age, role, company size), context (job-to-be-done, environment), needs/constraints"
  - "Gather through user interviews, surveys, data analysis"
  - "Map Behaviors: how often they use product, depth of usage, lifecycle stage"
  - "Assess Value: revenue contribution, strategic importance, cost/risk"
  - "Combine into segments and rank to create 3-5 distinct buyer personas, each representing high-value segment"

- **Q1559 (Segment users for e-commerce platform):** Focus on e-commerce context
  - "Define Personas: demographics (age, income, location), shopping context (occasion, urgency), needs (price-sensitive, quality-focused, convenience)"
  - "Analyze Behaviors: purchase frequency (daily/weekly/occasional), basket size (small/medium/large), product categories"
  - "Assess Value: revenue (AOV, LTV), strategic value (repeat purchases, referrals), cost (returns, support)"
  - "Combine into segments like 'Power Shoppers √ó High Frequency √ó High Value' and rank to focus on most valuable segment"

- **Q2147 (Segment customers for growth initiative):** Emphasize growth focus
  - "Identify high-growth-potential segments using Persona √ó Behavior √ó Value"
  - "Define Personas representing untapped markets or expansion opportunities"
  - "Analyze Behaviors indicating growth potential: new users, increasing engagement, expanding usage"
  - "Assess Value: high LTV, low CAC, strong network effects, strategic importance"
  - "Rank segments by growth potential (size √ó growth rate √ó value), focus on top segment for growth initiative"

- **Q690 (Design WhatsApp for 3 segments):** Focus on product design
  - "Use Persona √ó Behavior √ó Value to understand each segment"
  - "Senior citizens: Persona (age 65+, less tech-savvy, need simplicity), Behavior (occasional use, voice calls preferred), Value (family connection, low churn)"
  - "Families: Persona (parents, multi-generational, need coordination), Behavior (daily use, group chats, media sharing), Value (high engagement, network effects)"
  - "Businesses: Persona (small/medium businesses, need efficiency), Behavior (daily use, team coordination), Value (high LTV, expansion potential)"
  - "Rank segments and design for primary segment while ensuring secondary segments aren't excluded"
