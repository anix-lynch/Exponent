## üí° Data Scientist Role Overview

Data Scientists build predictive models and extract insights from data. Key skills:

- **Machine Learning**: Supervised, unsupervised, deep learning
- **Statistics & Experimentation**: A/B testing, hypothesis testing, probability
- **Data Analysis**: Exploratory, root cause, business metrics
- **Model Evaluation**: Metrics, cross-validation, bias-variance tradeoff
- **Feature Engineering**: Creating and selecting features
- **Model Deployment**: Production ML, monitoring, MLOps
- **Specialized Areas**: NLP, Computer Vision, Time Series, Recommendation Systems
