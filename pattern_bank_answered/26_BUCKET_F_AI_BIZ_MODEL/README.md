# AI Business Models Reference (2024)

**Purpose:** Understanding AI business models, metrics frameworks, and unit economics (NOT for interview prep)

**Structure:**
- `01_API_Infrastructure_Models.md` - API access, platforms, hardware
- `02_SaaS_Enterprise_Models.md` - AI SaaS, enterprise consulting, licensing
- `03_Consumer_Product_Models.md` - Creative tools, enhanced products, content
- `04_Token_Crypto_Models.md` - Token-based, decentralized AI
- `05_Industry_Specific_Models.md` - Healthcare, finance, data monetization

Each file includes:
- Business model description
- Metrics framework (NSM, ARR, KPIs, Leading Indicators, Guardrails)
- Unit economics
- ASCII tree visualization
- Company examples
