# AI Business Models: SaaS & Enterprise

**Category:** AI software-as-a-service, enterprise consulting, and licensing models

---

## 1. AI SaaS Model

**Business Model:** AI solutions integrated into industry-specific tools or platforms (subscription-based)

**Companies:**
- **DataRobot** - Automated machine learning platform
- **Suki AI** - AI assistant for healthcare documentation
- **Observe.AI** - AI for call center analysis and optimization
- **Gong** - AI-powered revenue intelligence
- **Cresta** - AI for contact centers

---

### Metrics Framework

```
AI SaaS Model Metrics
│
├─ NSM (North Star Metric)
│   └─ ARR (Annual Recurring Revenue)
│       └─ Or: NRR (Net Revenue Retention)
│
├─ Input KPIs
│   ├─ Revenue drivers
│   │   ├─ New customer acquisition
│   │   ├─ Expansion revenue (upsell, add-ons)
│   │   ├─ Contract renewals
│   │   └─ Price increases
│   │
│   ├─ Product adoption
│   │   ├─ Feature usage
│   │   ├─ Seat utilization
│   │   ├─ Integration adoption
│   │   └─ Workflow depth (daily active use)
│   │
│   └─ Customer health
│       ├─ Product engagement score
│       ├─ Support ticket volume
│       ├─ Customer satisfaction (NPS)
│       └─ Time to value
│
├─ Leading Indicators
│   ├─ Trial → paid conversion
│   ├─ Feature adoption rate
│   ├─ Expansion signal strength
│   └─ Product-qualified leads (PQLs)
│
└─ Guardrails
    ├─ Churn rate (gross/net)
    ├─ CAC payback period
    ├─ Customer concentration risk
    └─ LTV:CAC ratio
```

---

### Unit Economics

**Key Metrics:**
- **ARR per customer:** Average contract value × customers
- **CAC:** Sales & marketing cost / New customers
- **LTV:** Average revenue per customer × Gross margin % × Customer lifetime (months)
- **LTV:CAC ratio:** Target 3:1 or higher
- **CAC payback:** Months to recover acquisition cost

**Example (DataRobot-style):**
- **Average contract:** $50K-200K ARR
- **CAC:** $30K-50K (enterprise sales cycle)
- **Gross margin:** ~75-80% (software)
- **Churn:** ~5-10% annually
- **LTV:** $500K-1M (5-7 year lifetime)

**Optimization levers:**
- Product-led growth (self-serve trials)
- Expansion revenue (more seats, features)
- Customer success (reduce churn, increase NRR)

## 2. Enterprise AI Consulting Model

**Business Model:** Customized AI implementations and consulting for large organizations

**Companies:**
- **Fractal Analytics** - AI and analytics consulting
- **Zest AI** - AI-driven credit underwriting solutions
- **Palantir Technologies** - Data analytics platforms (government/commercial)
- **C3.ai** - Enterprise AI applications

---

### Metrics Framework

```
Enterprise Consulting Model Metrics
│
├─ NSM (North Star Metric)
│   └─ Total Contract Value (TCV)
│       └─ Or: Annual Services Revenue
│
├─ Input KPIs
│   ├─ Revenue drivers
│   │   ├─ New client wins
│   │   ├─ Contract renewals / extensions
│   │   ├─ Project expansion (scope creep)
│   │   └─ Multi-project relationships
│   │
│   ├─ Project health
│   │   ├─ Project margin
│   │   ├─ Billable utilization rate
│   │   ├─ Project delivery on-time
│   │   └─ Client satisfaction scores
│   │
│   └─ Customer health
│       ├─ Client retention rate
│       ├─ Project pipeline
│       ├─ Reference-ability
│       └─ Net Promoter Score (NPS)
│
├─ Leading Indicators
│   ├─ Proposal win rate
│   ├─ Pipeline conversion
│   ├─ Consultant utilization
│   └─ Client engagement depth
│
└─ Guardrails
    ├─ Project margin (target 20-30%)
    ├─ Consultant utilization (target 80%+)
    ├─ Client concentration risk
    └─ Revenue recognition timing
```

---

### Unit Economics

**Key Metrics:**
- **Revenue per consultant:** Billable hours × rate
- **Project margin:** Revenue - consultant costs - overhead
- **Utilization rate:** Billable hours / Total available hours
- **CAC:** Sales cost / New client wins

**Example (Fractal-style):**
- **Billable rate:** $200-400/hour (senior consultants)
- **Utilization:** 75-85% target
- **Project margin:** 25-35%
- **Sales cycle:** 6-12 months (enterprise)
- **Client lifetime:** 3-5 years (multiple projects)

**Optimization levers:**
- Productization (reusable frameworks, IP)
- Offshore delivery (cost arbitrage)
- Outcome-based pricing (value share)

## 3. AI Licensing Model

**Business Model:** License AI algorithms or models to other businesses

**Companies:**
- **Landing AI** - Vision systems for industrial applications
- **Pathmind** - AI for reinforcement learning (industry)
- **Cortica** - Autonomous AI platform licensing

---

### Metrics Framework

```
AI Licensing Model Metrics
│
├─ NSM (North Star Metric)
│   └─ Licensing Revenue (Annual)
│       └─ Or: Number of licensed deployments
│
├─ Input KPIs
│   ├─ Revenue drivers
│   │   ├─ New license agreements
│   │   ├─ License renewals
│   │   ├─ Per-unit royalties
│   │   └─ Multi-product licensing
│   │
│   ├─ Product adoption
│   │   ├─ Licensed deployments
│   │   ├─ Integration success rate
│   │   ├─ API/SDK usage
│   │   └─ Customer support tickets
│   │
│   └─ Customer health
│       ├─ License utilization
│       ├─ Renewal probability
│       ├─ Customer satisfaction
│       └─ Reference-ability
│
├─ Leading Indicators
│   ├─ Pilot program conversions
│   ├─ SDK downloads
│   ├─ Developer engagement
│   └─ Partnership pipeline
│
└─ Guardrails
    ├─ License churn rate
    ├─ Revenue concentration (top customers)
    ├─ IP protection / piracy
    └─ Competitive displacement
```

---

### Unit Economics

**Key Metrics:**
- **License revenue:** Upfront fee + ongoing royalties
- **Cost:** R&D amortization + support costs
- **Gross margin:** 80-90% (software/IP)
- **CAC:** Sales cost / New licenses

**Example (Landing AI-style):**
- **Upfront license:** $100K-500K
- **Royalty:** 5-10% of product revenue (if applicable)
- **Support cost:** ~10-15% of license value
- **Gross margin:** ~85%

**Optimization levers:**
- Volume licensing (enterprise deals)
- SaaS conversion (recurring revenue)
- White-label partnerships

---

## Key Differences

| Model | Revenue Model | Sales Cycle | Margin Profile |
|-------|--------------|-------------|----------------|
| **AI SaaS** | Recurring subscription | 1-6 months | High margin (75-80%) |
| **Enterprise Consulting** | Project-based | 6-12 months | Medium margin (25-35%) |
| **Licensing** | Upfront + royalties | 3-9 months | Very high margin (80-90%) |

---

## Common Patterns

1. **Land and Expand:** Start with pilot/project → expand to full deployment
2. **Productization:** Turn consulting IP into repeatable SaaS products
3. **Outcome-based pricing:** Share in customer value creation
4. **Platform play:** Build ecosystem around core AI capability
